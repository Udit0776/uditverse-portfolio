# UditVerse World Map

## Overview

UditVerse is an interactive portfolio experience designed as a bright and hopeful mansion that represents my journey as a developer, creator, and lifelong learner.

Visitors explore the mansion instead of navigating traditional portfolio pages.

Every room tells part of my story.
